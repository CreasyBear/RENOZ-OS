## Summary

- Audit scope: actionable TODO/placeholder items in `src/server`, `src/routes`, `src/hooks`, `src/components`, and `src/trigger`.
- Triage scale:
  - `P0`: fake-success or incorrect persisted state
  - `P1`: user-visible unavailable path or indefinite async path
  - `P2`: honest explicit unavailable contract
  - `P3`: internal enhancement or implementation note with no immediate user risk
- Dispositions:
  - `Implement as real workflow`
  - `Re-contract temporarily`
  - `Leave visible with explicit unavailable state`
  - `Defer`
