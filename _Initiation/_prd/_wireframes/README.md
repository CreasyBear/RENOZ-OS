# Wireframes Directory

> **ASCII Wireframes for Renoz CRM PRD Stories**
> **Location**: `memory-bank/prd/_wireframes/`

---

## Purpose

This directory contains ASCII wireframes that visually document UI layouts, component structures, and interaction flows for PRD stories. Wireframes serve as:

1. **Design Specifications** - Clear visual contracts for implementation
2. **Developer Guidance** - Reduce ambiguity in UI requirements
3. **Review Artifacts** - Enable visual review before coding begins
4. **Documentation** - Capture UI decisions alongside requirements
