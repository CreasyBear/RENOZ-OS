# Wireframe: DOM-SET-007 - Unified Integrations Hub

## Story Reference

- **Story ID**: DOM-SET-007
- **Name**: Create Unified Integrations Hub
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Settings page with Card grid
- **Component Type**: Integration status dashboard

## Overview

Central page showing all system integrations with connection status, configuration links, and test connection capabilities. Features cards for Xero, Resend, and Trigger.dev with status indicators, recent activity, and quick actions.

## UI Patterns (Reference Implementation)

### Card (Integration Cards)
- **Pattern**: RE-UI Card
- **Reference**: `_reference/.reui-reference/registry/default/ui/card.tsx`
- **Features**:
  - Card layout for each integration (Xero, Resend, Trigger.dev)
  - Logo image in card header
  - Status details and recent activity in card body
  - Action buttons in card footer

### Badge (Status Indicator)
- **Pattern**: RE-UI Badge
- **Reference**: `_reference/.reui-reference/registry/default/ui/badge.tsx`
- **Features**:
  - Status badge showing Connected, Disconnected, Error, Testing states
  - Color-coded (green for connected, red for error, amber for testing)
  - Dot indicator with text label

### Button (Test Connection & Configure)
- **Pattern**: RE-UI Button
- **Reference**: `_reference/.reui-reference/registry/default/ui/button.tsx`
- **Features**:
  - Test Connection button with loading state during test
  - Configure button linking to integration settings
  - View Stats/View Runs external links with arrow icon

### Alert (Connection Status Messages)
- **Pattern**: RE-UI Alert
- **Reference**: `_reference/.reui-reference/registry/default/ui/alert.tsx`
- **Features**:
  - Success alert showing test connection passed with latency
  - Error alert showing authentication expired with reconnect action
  - Warning alert for disconnected state
  - Auto-dismiss success after 5 seconds

### DataTable (Activity Log)
- **Pattern**: RE-UI DataTable
- **Reference**: `_reference/.reui-reference/registry/default/example/data-table-demo.tsx`
- **Features**:
  - Desktop-only table showing recent integration activity
  - Columns: Time, Integration, Event, Status
  - Link to view all activity
  - Real-time updates for recent events

## Accessibility Notes

### Focus Order

1. **Page Load**
   - Focus on first integration card
   - Skip link to specific integration available

2. **Card Navigation**
   - Tab through cards
   - Within card: Test button -> Configure/Action link
   - Status badge is not focusable (read by screen reader as part of card)

3. **Test Connection Flow**
   - Focus stays on Test button during test
   - Result announced via live region
   - Error moves focus to action button if available

## Animation Choreography

### Page Load

```
CARDS ENTRY:
- Duration: 400ms total
- Stagger: 100ms per card
- Each card:
  - Opacity: 0 -> 1
  - Transform: translateY(16px) -> translateY(0)
  - Scale: 0.98 -> 1
```

### Status Check

```
STATUS DOT PULSE (Checking):
- Duration: 1s
- Opacity: 1 -> 0.5 -> 1
- Loop: continuous until result
- Color: amber during check
```

### Test Connection

```
TEST BUTTON LOADING:
- Duration: continuous
- Spinner rotation: 1s linear infinite
- Text: "Test Connection" -> "Testing..."

TEST SUCCESS:
- Duration: 500ms
- Status dot: pulse green (200ms)
- Success message: fade in (200ms)
- After 5s: fade out (300ms)

TEST ERROR:
- Duration: 500ms
- Status dot: flash red
- Error message: slide in (200ms)
- Shake animation: subtle (200ms)
```

### Status Change

```
CONNECTED -> DISCONNECTED:
- Duration: 300ms
- Status dot: color transition
- Card: subtle red tint flash

DISCONNECTED -> CONNECTED:
- Duration: 400ms
- Status dot: scale bounce (0.8 -> 1.2 -> 1)
- Card: subtle green tint flash
- Stats: fade in (200ms)
```

### Card Interactions

```
HOVER:
- Duration: 150ms
- Box shadow: elevate slightly
- Transform: translateY(-2px)

CLICK/ACTIVE:
- Duration: 100ms
- Transform: scale(0.98)
```
