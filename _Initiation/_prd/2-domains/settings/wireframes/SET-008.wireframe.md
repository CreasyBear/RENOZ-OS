# Wireframe: DOM-SET-008 - Settings Search

## Story Reference

- **Story ID**: DOM-SET-008
- **Name**: Add Settings Search
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Search Input with Dropdown Results
- **Component Type**: Global settings search with navigation

## Overview

Search functionality across all settings sections including section names, field labels, and setting descriptions. Features debounced search, results dropdown with highlighting, scroll-to and highlight on navigation, recent searches saved to localStorage, and keyboard shortcuts (Cmd/Ctrl+K).

## UI Patterns (Reference Implementation)

### Input (Search Combobox)
- **Pattern**: RE-UI Input
- **Reference**: `_reference/.reui-reference/registry/default/ui/input.tsx`
- **Features**:
  - Search input with magnifying glass icon prefix
  - Clear button (X) appears when text entered
  - Debounced input to avoid excessive searches
  - Keyboard shortcut hint (âŒ˜K) shown in placeholder

### Popover (Search Results Dropdown)
- **Pattern**: RE-UI Popover
- **Reference**: `_reference/.reui-reference/registry/default/ui/popover.tsx`
- **Features**:
  - Dropdown below search input showing results
  - Grouped results by settings section
  - Keyboard navigation with arrow keys
  - Enter to navigate to selected result

### Command (Keyboard Shortcuts)
- **Pattern**: RE-UI Command
- **Reference**: `_reference/.reui-reference/registry/default/ui/command.tsx`
- **Features**:
  - Cmd/Ctrl+K to open search from anywhere
  - Arrow keys to navigate results
  - Enter to select, Escape to close
  - Tab to navigate to recent searches section

### Badge (Recent Search Chips)
- **Pattern**: RE-UI Badge
- **Reference**: `_reference/.reui-reference/registry/default/ui/badge.tsx`
- **Features**:
  - Clickable chips showing recent searches
  - X button on hover to remove individual search
  - Clear All button to remove all recent searches
  - Saved to localStorage for persistence

### Separator
- **Pattern**: RE-UI Separator
- **Reference**: `_reference/.reui-reference/registry/default/ui/separator.tsx`
- **Features**:
  - Horizontal divider between results and recent searches
  - Visual section separation in dropdown
  - Improves scannability of search UI

## Accessibility Notes

### Focus Order

1. **Keyboard Shortcut**
   - Cmd/Ctrl+K opens search from anywhere in settings
   - Focus moves to search input

2. **Search Input**
   - Focus on input
   - Type to search
   - Results appear below

3. **Results Navigation**
   - Arrow Down: Move to first result / next result
   - Arrow Up: Previous result / back to input
   - Enter: Navigate to selected result
   - Escape: Close dropdown, clear search

4. **Recent Searches**
   - Tab to navigate recent search chips
   - Enter/Space to activate
   - Tab to Clear All

## Animation Choreography

### Search Open

```
DROPDOWN OPEN:
- Duration: 200ms
- Easing: ease-out
- Transform: translateY(-8px) -> translateY(0)
- Opacity: 0 -> 1
- Scale: 0.98 -> 1

COMMAND PALETTE OPEN (Optional):
- Duration: 200ms
- Backdrop: fade in (150ms)
- Panel: scale(0.95) -> scale(1)
- Opacity: 0 -> 1
```

### Search Input

```
TYPING INDICATOR:
- Duration: continuous
- Spinner: rotation 1s linear
- Shows during debounce period

CLEAR BUTTON APPEAR:
- Duration: 150ms
- Opacity: 0 -> 1
- Scale: 0.8 -> 1
```

### Results

```
RESULTS ENTRY:
- Duration: 200ms
- Stagger: 30ms per result
- Each result: translateY(-4px) -> translateY(0)
- Opacity: 0 -> 1

RESULT HIGHLIGHT (Selected):
- Duration: 100ms
- Background: transition to highlight color

MATCH TEXT HIGHLIGHT:
- Bold matched characters immediately (no animation)
```

### Navigation & Highlight

```
SCROLL TO FIELD:
- Duration: 300ms
- Easing: ease-out
- Smooth scroll to field position

FIELD HIGHLIGHT:
- Duration: 3000ms total
- Entry: opacity 0 -> 1 (200ms)
- Pulse: scale 1 -> 1.01 -> 1, 2 times (600ms each)
- Exit: opacity 1 -> 0 (500ms)
- Background: amber/yellow highlight
```

### Recent Searches

```
CHIP CLICK:
```
