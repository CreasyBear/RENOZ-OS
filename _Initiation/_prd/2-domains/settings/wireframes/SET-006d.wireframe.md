# Wireframe: DOM-SET-006d - Integrate Custom Fields into Customers

## Story Reference

- **Story ID**: DOM-SET-006d
- **Name**: Integrate Custom Fields into Customers
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Form Section Integration
- **Component Type**: Custom fields integration in customer forms and detail view

## Overview

Integrate CustomFieldsFormSection component into customer create/edit forms and detail page. Features collapsible section with expand/collapse toggle, proper loading states, and error handling isolated from main form.

## UI Patterns (Reference Implementation)

### Collapsible (Custom Fields Section)
- **Pattern**: RE-UI Collapsible
- **Reference**: `_reference/.reui-reference/registry/default/ui/collapsible.tsx`
- **Features**:
  - Expand/collapse toggle for custom fields section
  - Shows field count badge when collapsed
  - Arrow icon indicates current state (â–¶ collapsed, â–¼ expanded)
  - Default collapsed on mobile, expanded on desktop

### Card (Custom Fields Container)
- **Pattern**: RE-UI Card
- **Reference**: `_reference/.reui-reference/registry/default/ui/card.tsx`
- **Features**:
  - Card wraps custom fields section for visual grouping
  - Border styling to separate from main form sections
  - Header with title and expand/collapse control

### Form Fields (Dynamic Inputs)
- **Pattern**: RE-UI Input, Select, Checkbox
- **Reference**: `_reference/.reui-reference/registry/default/ui/input.tsx`
- **Features**:
  - Text input for Account Manager field
  - Select dropdown for Industry field with required validation
  - Date picker for Contract End Date
  - Checkbox for VIP Customer boolean field

### Alert (Error Boundary)
- **Pattern**: RE-UI Alert
- **Reference**: `_reference/.reui-reference/registry/default/ui/alert.tsx`
- **Features**:
  - Non-blocking error alert when custom fields fail to load
  - Retry button to attempt reload
  - Continue Without button to proceed with main form
  - Info alert explaining main form continues to work

### Skeleton (Loading State)
- **Pattern**: RE-UI Skeleton
- **Reference**: `_reference/.reui-reference/registry/default/ui/skeleton.tsx`
- **Features**:
  - Skeleton placeholders for custom fields while loading
  - Shimmer animation during fetch
  - Maintains layout to prevent content shift

## Accessibility Notes

### Focus Order

1. **Create/Edit Form**
   - Tab through basic fields first
   - Tab to custom fields section header
   - Enter/Space to expand/collapse
   - When expanded, tab through custom fields
   - Continue to address fields after

2. **Expand/Collapse**
   - Button is focusable
   - Space/Enter toggles
   - Arrow indicates state
   - Content hidden from tab order when collapsed

3. **Detail Page**
   - Edit button is focusable
   - Custom field values are not interactive (display only)

## Animation Choreography

### Expand/Collapse

```
EXPAND:
- Duration: 300ms
- Easing: ease-out
- Content height: 0 -> auto (use max-height)
- Opacity: 0 -> 1
- Arrow rotation: 0deg -> 90deg

COLLAPSE:
- Duration: 250ms
- Easing: ease-in
- Content height: auto -> 0
- Opacity: 1 -> 0
- Arrow rotation: 90deg -> 0deg
```

### Fields Entry (When Expanded)

```
STAGGER ENTRY:
- Duration: 300ms total
- Stagger: 50ms per field
- Each field: translateY(8px) -> translateY(0)
- Opacity: 0 -> 1
```

### Focus First Field

```
FOCUS ANIMATION:
- Delay: 300ms (after expand complete)
- Duration: 200ms
- First field receives focus
- Ring animation draws attention
```

### Error State

```
ERROR ENTRY:
- Duration: 250ms
- Transform: scale(0.98) -> scale(1)
- Opacity: 0 -> 1
- Background: subtle red tint
```

## Integration Example

```typescript
// Customer create form
function CustomerCreateForm() {
  const form = useForm<CustomerFormData>({
    defaultValues: {
      name: '',
      email: '',
    },
  });

  const customFields = useCustomerCustomFields({
    defaultExpanded: false, // Collapsed on mobile
  });

  const handleSubmit = async (data: CustomerFormData) => {
    // Validate custom fields
    if (!customFields.validate()) {
      customFields.setIsExpanded(true); // Expand to show errors
      return;
    }

    try {
      const customer = await createCustomer({
        ...data,
        customFieldValues: customFields.getSubmitData(),
      });

      toast.success('Customer created');
      onSuccess(customer);
    } catch (error) {
      toast.error('Failed to create customer');
    }
  };

  return (
    <Form {...form} onSubmit={handleSubmit}>
      <FormSection title="Basic Information">
        <FormField name="name" label="Business Name" required />
        <FormField name="email" label="Email" required />
        <FormField name="phone" label="Phone" />
      </FormSection>

      {/* Only render if fields are configured */}
      {customFields.fields.length > 0 && (
        <CollapsibleCustomFieldsSection
          entityType="customers"
          values={customFields.values}
          onChange={customFields.setValues}
          errors={customFields.errors}
          isExpanded={customFields.isExpanded}
          onToggle={customFields.toggleExpanded}
          isLoading={customFields.isLoadingFields}
          error={customFields.fieldsError}
          onRetry={customFields.retry}
        />
      )}

      <FormSection title="Address">
        {/* Address fields */}
      </FormSection>

      <FormFooter>
        <Button variant="secondary" onClick={onCancel}>
          Cancel
        </Button>
        <Button type="submit">
          Create Customer
        </Button>
      </FormFooter>
    </Form>
  );
}
```
