# Wireframe: DOM-SET-006b - Custom Fields Configuration UI

## Story Reference

- **Story ID**: DOM-SET-006b
- **Name**: Custom Fields Configuration UI
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Settings page with Select, DataTable, and Dialog
- **Component Type**: Admin CRUD interface for custom field definitions

## Overview

Admin UI to define custom fields per entity type. Features entity selector dropdown, DataTable of existing fields with name/type/required status, add/edit/delete dialogs, and drag-and-drop reordering for field display order.

## UI Patterns (Reference Implementation)

### Select (Entity Type Dropdown)
- **Pattern**: RE-UI Select
- **Reference**: `_reference/.reui-reference/registry/default/ui/select.tsx`
- **Features**:
  - Dropdown to switch between Customers, Orders, Quotes entities
  - Icons for each entity type for visual recognition
  - Loads corresponding custom fields when selection changes

### DataTable (Fields List)
- **Pattern**: RE-UI DataTable
- **Reference**: `_reference/.reui-reference/registry/default/example/data-table-demo.tsx`
- **Features**:
  - Table showing field name, type, and required status
  - Drag handle for reordering fields
  - Edit and Delete action buttons per row
  - Drag-and-drop reordering with visual feedback

### Dialog (Add/Edit Field)
- **Pattern**: RE-UI Dialog
- **Reference**: `_reference/.reui-reference/registry/default/ui/dialog.tsx`
- **Features**:
  - Modal dialog for adding or editing custom field definition
  - Form with field name, type dropdown, required checkbox
  - Type-specific options (select options, number min/max, date format)
  - Save and Cancel actions

### RadioGroup (Field Type Selection)
- **Pattern**: RE-UI RadioGroup
- **Reference**: `_reference/.reui-reference/registry/default/ui/radio-group.tsx`
- **Features**:
  - Radio buttons for field type (Text, Number, Date, Select, Checkbox)
  - Visual icons for each field type
  - Shows additional config fields based on selected type

### Badge (Required Indicator)
- **Pattern**: RE-UI Badge
- **Reference**: `_reference/.reui-reference/registry/default/ui/badge.tsx`
- **Features**:
  - Visual badge showing "Required" or "Optional" status
  - Color-coded (red for required, gray for optional)
  - Appears in both table and edit dialog

## Accessibility Notes

### Focus Order

1. **Page Load**
   - Focus on entity selector dropdown
   - Skip link to Add Field button

2. **Entity Selection**
   - Tab to dropdown, arrow keys to select
   - Selection loads fields for that entity

3. **Table Navigation**
   - Tab through table rows
   - Each row: drag handle -> Edit button -> Delete button
   - Enter on Edit opens dialog

4. **Drag and Drop**
   - Drag handles are focusable
   - Space/Enter to pick up
   - Arrow keys to move
   - Space/Enter to drop
   - Escape to cancel

5. **Dialog Navigation**
   - Focus trapped when open
   - Tab through: Name -> Type -> Options -> Required -> Cancel -> Submit
   - Escape to close

## Animation Choreography

### Entity Switch

```
ENTITY CHANGE:
- Duration: 300ms
- Table fade out: 150ms
- Table fade in with new data: 150ms
- Stagger rows: 30ms per row
```

### Field Operations

```
ADD FIELD ROW:
- Duration: 300ms
- Transform: translateY(-10px) -> translateY(0)
- Opacity: 0 -> 1
- Background: highlight flash (200ms)

DELETE FIELD ROW:
- Duration: 300ms
- Transform: scale(1) -> scale(0.95)
- Opacity: 1 -> 0
- Height: collapse to 0
- Remaining rows: slide up (200ms)
```

### Drag and Drop

```
PICK UP:
- Duration: 150ms
- Transform: scale(1) -> scale(1.02)
- Box shadow: elevated
- Opacity: 1 -> 0.9
- Cursor: grabbing

DRAG:
- Transform: follow cursor
- Drop indicator: fade in at target location

DROP:
- Duration: 200ms
- Transform: scale(1.02) -> scale(1)
- Box shadow: reset
- Opacity: 0.9 -> 1
- Other rows: animate to new positions (300ms)

CANCEL:
- Duration: 200ms
- Return to original position
- Reset all visual states
```

### Dialog Animations

```
OPEN:
- Duration: 250ms
- Backdrop: fade in (150ms)
- Panel: scale(0.95) -> scale(1)
- Opacity: 0 -> 1

CLOSE:
- Duration: 200ms
- Panel: opacity 1 -> 0
- Backdrop: fade out
```

### Options List (Select Type)

```
ADD OPTION:
- Duration: 200ms
- Transform: translateY(-8px) -> translateY(0)
- Opacity: 0 -> 1

REMOVE OPTION:
- Duration: 200ms
- Transform: scale(1) -> scale(0.95)
- Opacity: 1 -> 0
- Height collapse
```
