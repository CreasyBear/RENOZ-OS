# Wireframe: DOM-SET-005b - Business Hours Settings Page

## Story Reference

- **Story ID**: DOM-SET-005b
- **Name**: Business Hours Settings Page
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Settings Form with Grid and DataTable
- **Component Type**: Weekly schedule grid with time inputs and holidays table

## Overview

Admin UI for configuring business hours including weekly schedule grid with TimeInput pickers, day enable/disable toggles, timezone selection, and holidays management with add/delete functionality.

## UI Patterns (Reference Implementation)

### Switch (Day Enable/Disable)
- **Pattern**: RE-UI Switch
- **Reference**: `_reference/.reui-reference/registry/default/ui/switch.tsx`
- **Features**:
  - Toggle switch to enable/disable each day of the week
  - Disabled state grays out time pickers for that day
  - Clear visual feedback for on/off state

### Input (Time Picker)
- **Pattern**: RE-UI Input
- **Reference**: `_reference/.reui-reference/registry/default/ui/input.tsx`
- **Features**:
  - Time input with 12hr format (9:00 AM - 5:00 PM)
  - Validation to ensure end time is after start time
  - Disabled state when day toggle is off

### Select (Timezone Dropdown)
- **Pattern**: RE-UI Select
- **Reference**: `_reference/.reui-reference/registry/default/ui/select.tsx`
- **Features**:
  - Searchable timezone dropdown with IANA timezones
  - Groups timezones by region (Australia, Pacific, etc.)
  - Shows current time in selected timezone

### DataTable (Holidays List)
- **Pattern**: RE-UI DataTable
- **Reference**: `_reference/.reui-reference/registry/default/example/data-table-demo.tsx`
- **Features**:
  - Table of public holidays with date and name
  - Add/delete actions per row
  - Sortable by date column

### Dialog (Add Holiday)
- **Pattern**: RE-UI Dialog
- **Reference**: `_reference/.reui-reference/registry/default/ui/dialog.tsx`
- **Features**:
  - Modal dialog for adding new holiday
  - Date picker for holiday date selection
  - Text input for holiday name
  - Save and Cancel actions

## Accessibility Notes

### Focus Order

1. **Page Load**
   - Focus on timezone select
   - Skip link to Save button available

2. **Schedule Grid Navigation**
   - Tab: Timezone -> Monday toggle -> Monday start -> Monday end -> Tuesday toggle -> ...
   - Continue through all days in order
   - Tab to Save button after Sunday

3. **Day Toggle Interaction**
   - Space/Enter to toggle day on/off
   - When disabled, time inputs are removed from tab order

4. **Holidays Section**
   - Tab to Add Holiday button
   - Tab through table rows and Delete buttons
   - Enter on Add opens dialog

5. **Holiday Dialog**
   - Focus trapped in dialog
   - Tab: Date input -> Name input -> Quick add buttons -> Cancel -> Add
   - Escape to close

## Animation Choreography

### Day Toggle

```
ENABLE DAY:
- Duration: 300ms
- Toggle slide: left to right (150ms)
- Time inputs: fade in + slide down (150ms, staggered)
- Color: transition to active color (200ms)

DISABLE DAY:
- Duration: 250ms
- Time inputs: fade out + slide up (150ms)
- Toggle slide: right to left (100ms)
- Color: transition to muted (200ms)
```

### Time Input Changes

```
VALUE CHANGE:
- Duration: 200ms
- Number: counter animation if changed by arrows
- Border: highlight flash (100ms)

VALIDATION ERROR:
- Duration: 400ms
- Shake: translateX (-4, 4, -4, 4, 0)
- Border: transition to error color (150ms)
```

### Holiday Management

```
ADD HOLIDAY ROW:
- Duration: 300ms
- Transform: translateY(-10px) -> translateY(0)
- Opacity: 0 -> 1
- Background: highlight flash (200ms)

DELETE HOLIDAY ROW:
- Duration: 250ms
- Transform: scale(1) -> scale(0.95)
- Opacity: 1 -> 0
- Height: collapse to 0 (150ms)
- Remaining rows: slide up (200ms)

QUICK ADD BUTTON:
- Duration: 150ms
- Scale: 1 -> 0.95 -> 1 on click
- Background: ripple effect
```

### Dialog Animations

```
DIALOG OPEN:
- Duration: 250ms
- Backdrop: fade in (150ms)
- Panel: scale(0.95) -> scale(1), translateY(10px) -> translateY(0)
- Opacity: 0 -> 1

DIALOG CLOSE:
- Duration: 200ms
- Panel: opacity 1 -> 0
- Backdrop: fade out (200ms)
```

### Save Animation

```
SAVE SUCCESS:
- Duration: 400ms
- Button: text -> checkmark icon (200ms)
- Background: pulse green (200ms)
- Text: "Save" -> "Saved" -> "Save" (hold 1s)
```
