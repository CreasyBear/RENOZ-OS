# Wireframe: DOM-SET-002 - Enhanced Audit Log Viewer

## Story Reference

- **Story ID**: DOM-SET-002
- **Name**: Enhance Audit Log Viewer
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: DataTable with FilterBar
- **Component Type**: List page with filters, search, and export

## Overview

Enhanced audit log page with search, user filter, date range filter, entity type filter, and CSV export. Admin-only access with responsive design featuring mobile filter drawer and desktop inline filters.

## UI Patterns (Reference Implementation)

### DataTable
- **Pattern**: RE-UI DataTable
- **Reference**: `_reference/.reui-reference/registry/default/example/data-table-demo.tsx`
- **Features**:
  - Sortable columns for timestamp, action, entity, and user
  - Row selection with click-to-view detail behavior
  - Pagination with showing X of Y records display
  - Responsive design with mobile card view and desktop table view

### Input (Search)
- **Pattern**: RE-UI Input
- **Reference**: `_reference/.reui-reference/registry/default/ui/input.tsx`
- **Features**:
  - Search icon prefix for audit record search
  - Debounced search with loading spinner during query
  - Clear button (X) when search has text

### Select (Filter Dropdowns)
- **Pattern**: RE-UI Select
- **Reference**: `_reference/.reui-reference/registry/default/ui/select.tsx`
- **Features**:
  - User dropdown with all users list
  - Entity type dropdown with icons for each type
  - Action type dropdown for created/updated/deleted

### Sheet (Mobile Filter Drawer)
- **Pattern**: RE-UI Sheet
- **Reference**: `_reference/.reui-reference/registry/default/ui/sheet.tsx`
- **Features**:
  - Slide-in drawer from bottom on mobile for filters
  - Apply and Clear All actions at bottom
  - Close button in header

### Badge (Active Filter Chips)
- **Pattern**: RE-UI Badge
- **Reference**: `_reference/.reui-reference/registry/default/ui/badge.tsx`
- **Features**:
  - Shows active filters as removable chips
  - X button on each chip to remove individual filter
  - Count badge on Filter button showing number of active filters

## Accessibility Notes

### Focus Order

1. **Page Load**
   - Focus moves to search input
   - Skip link to main content available

2. **Filter Navigation**
   - Tab: Search -> Filter dropdowns -> Clear button -> Export button
   - Tab: Continue to table rows
   - Arrow keys navigate within filter dropdowns

3. **Table Navigation**
   - Tab through table rows
   - Enter to view details
   - Arrow keys for row navigation (optional enhancement)

4. **Detail Panel**
   - Focus trapped in panel when open
   - Escape to close
   - Tab through panel content

## Animation Choreography

### Page Load

```
STAGGER ENTRY:
- Duration: 400ms total
- Sequence:
  1. Header fades in (0-100ms)
  2. Search/filters slide in (100-200ms)
  3. Table rows stagger in (200-400ms)
     - Each row: 50ms delay, translateY(8px) -> 0
```

### Filter Interactions

```
DROPDOWN OPEN:
- Duration: 200ms
- Easing: ease-out
- Transform: scale(0.95) -> scale(1)
- Opacity: 0 -> 1

ACTIVE FILTER CHIP ENTRY:
- Duration: 200ms
- Transform: scale(0) -> scale(1)
- Opacity: 0 -> 1

FILTER CHIP REMOVAL:
- Duration: 150ms
- Transform: scale(1) -> scale(0)
- Opacity: 1 -> 0
- Remaining chips shift: 200ms ease-out

TABLE REFRESH (Filter Change):
- Duration: 300ms
- Old rows: fade out (150ms)
- New rows: stagger in (150ms)
```

### Detail Panel

```
PANEL SLIDE IN (Desktop):
- Duration: 300ms
- Easing: ease-out
- Transform: translateX(100%) -> translateX(0)
- Main content: width transition (300ms)

PANEL SLIDE OUT:
- Duration: 250ms
- Easing: ease-in
- Transform: translateX(0) -> translateX(100%)

MOBILE DETAIL VIEW:
- Duration: 300ms
- Full screen slide from right
- Backdrop: fade in 200ms
```

### Export Progress

```
PROGRESS BAR:
- Duration: continuous
- Width: 0% -> actual progress
- Easing: linear updates every 500ms

COMPLETE CHECKMARK:
- Duration: 400ms
- Scale: 0 -> 1.2 -> 1
- Color: transition to green (200ms)

DOWNLOAD BUTTON PULSE:
- Duration: 1s
- Opacity: 1 -> 0.7 -> 1
- Loop: 3 times to draw attention
```
