# Wireframe: DOM-SET-003b - Export Page UI

## Story Reference

- **Story ID**: DOM-SET-003b
- **Name**: Create Export Page UI
- **PRD**: memory-bank/prd/domains/settings.prd.json
- **Type**: Form with CheckboxGroup and DataTable
- **Component Type**: Settings page for data export

## Overview

Settings page for initiating and viewing data exports. Features entity selection via CheckboxGroup, format selection via RadioGroup, background job triggering, and export history table with download links.

## UI Patterns (Reference Implementation)

### Checkbox Group (Entity Selection)
- **Pattern**: RE-UI Checkbox
- **Reference**: `_reference/.reui-reference/registry/default/ui/checkbox.tsx`
- **Features**:
  - Multi-select checkboxes for exporting multiple entity types
  - Select All / Deselect All toggle for quick selection
  - Grouped by category (Core Data, Financial, Service)

### RadioGroup (Format Selection)
- **Pattern**: RE-UI RadioGroup
- **Reference**: `_reference/.reui-reference/registry/default/ui/radio-group.tsx`
- **Features**:
  - Card-style radio buttons for CSV vs JSON format
  - Visual representation of file format with icons
  - Additional options show conditionally (CSV delimiter, JSON indentation)

### DataTable (Export History)
- **Pattern**: RE-UI DataTable
- **Reference**: `_reference/.reui-reference/registry/default/example/data-table-demo.tsx`
- **Features**:
  - Table showing past exports with status, date, file size
  - Download button for completed exports
  - Status badges (Pending, Processing, Complete, Failed)

### Button (Export Actions)
- **Pattern**: RE-UI Button
- **Reference**: `_reference/.reui-reference/registry/default/ui/button.tsx`
- **Features**:
  - Primary button for "Start Export" action
  - Loading state during export generation
  - Success state transitions to "Download" button

### Progress Bar
- **Pattern**: RE-UI Progress
- **Reference**: `_reference/.reui-reference/registry/default/ui/progress.tsx`
- **Features**:
  - Real-time export generation progress (0-100%)
  - Shows estimated time remaining
  - Indeterminate mode for queued jobs

## Accessibility Notes

### Focus Order

1. **Page Load**
   - Focus on first entity checkbox
   - Skip link to export button available

2. **Entity Selection**
   - Tab through checkboxes
   - Space to toggle
   - Select All / Clear as shortcuts

3. **Format Selection**
   - Tab to radio group
   - Arrow keys to select format

4. **Export Action**
   - Tab to Start Export button
   - Enter to initiate

5. **History Table**
   - Tab through download/retry buttons
   - Enter to activate

## Animation Choreography

### Checkbox Selection

```
CHECKBOX TOGGLE:
- Duration: 150ms
- Easing: ease-out
- Checkmark: scale(0) -> scale(1.1) -> scale(1)
- Background: fade to checked color (100ms)

SELECT ALL:
- Duration: 300ms
- Stagger: 50ms per checkbox
- Each checkbox animates in sequence
```

### Export Progress

```
PROGRESS BAR:
- Duration: continuous
- Width: animated to current percentage
- Easing: linear updates every 500ms
- Pulse: subtle glow animation during processing

PROGRESS INCREMENT:
- Duration: 300ms per update
- Counter: number animation from old to new value
- Easing: ease-out
```

### Export Complete

```
SUCCESS ANIMATION:
- Duration: 500ms
- Progress bar: fill to 100% (200ms)
- Checkmark: scale(0) -> scale(1.2) -> scale(1) (300ms)
- Background: green pulse (200ms)

TOAST ENTRY:
- Duration: 250ms
- Transform: translateY(-20px) -> translateY(0)
- Opacity: 0 -> 1
```

### History Table

```
NEW EXPORT ROW:
- Duration: 300ms
- Transform: translateY(-10px) -> translateY(0)
- Opacity: 0 -> 1
- Background: highlight flash (200ms)

STATUS UPDATE:
- Duration: 200ms
- Old status: fade out (100ms)
- New status: fade in with bounce (100ms)
```

### Download Button

```
DOWNLOAD AVAILABLE:
- Duration: 1s
- Animation: pulse glow
- Loop: 3 times to draw attention
- Stops after first click

DOWNLOAD CLICKED:
- Duration: 200ms
- Scale: 1 -> 0.95 -> 1
- Background: ripple effect
```
